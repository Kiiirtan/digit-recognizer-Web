Under the Hood

Client-Side Data Processing

Before the neural network can analyze the drawing, the raw canvas data must be isolated, scaled, and transformed into a tensor matching the training dataset.

1

Bounding Box Extraction

Scans the RGBA pixel array to find the exact boundaries of the drawn stroke, cropping out all dead whitespace.

2

Dimensionality Reduction

The cropped segment is redrawn onto a hidden 28x28 pixel canvas, downsampling the high-resolution stroke into a low-resolution matrix.

Visualization of 28x28 downsampling

3

Array Serialization

Extracts the grayscale intensity of each pixel, forming a 2D array of 28x28 integers to be sent via HTTP POST.
